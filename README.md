## Overview

An AI-powered React web dashboard designed to optimize LinkedIn profiles for ATS systems and recruiter visibility.
The application generates optimized headlines, summaries, and experience bullet points using Google Gemini AI and evaluates profile strength using a deterministic scoring engine.


## Features

* AI-generated LinkedIn headlines and summaries
* Profile strength scoring system
* ATS keyword optimization
* Experience bullet point enhancement
* Secure client-side API key configuration
* Real-time profile analysis and suggestions
* Fully responsive UI design
